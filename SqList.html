SqList

Description

SqList是一个C++顺序表类，可以实现创建顺序表、增、删、查、输出等功能。

SqList类在表满的时候能够实现自动扩容，目前默认的容量是50，每次扩容增加50。

SqList中默认定义ElemType为char，可以自行修改。若定义为自定义的数据类型，须重载运算符‘==’。

Methods

SqList::SqList()

构造函数，初始化顺序表
bool SqList::isEmpty()

返回当前顺序表是否为空
int SqList::getLength()

返回顺序表的长度
void SqList::disList()

以每行10个，两个元素之间用空格分隔的方式输出顺序表
bool SqList::getElement(int index, ElemType e)

获取物理下标为index位置的元素，元素返回值保存在e。当物理下标超过顺序表长度或者小于1时，返回false；正确获取元素时返回true
int SqList::locateElement(ElemType e)

在顺序表中定位元素e，函数返回元素e在顺序表中的物理位序。当顺序表中不存在该元素时返回0
bool SqList::insertElement(int index, ElemType e)

在物理下标为index位置上插入元素e，函数返回插入操作是否成功。当物理下标超过顺序表长度或者小于1时，返回false；正确插入元素时返回true
bool SqList::deleteElement(int index, ElemType e)

删除物理下标为index位置的元素，并把该位置的元素赋给e。当物理下标超过顺序表长度或者小于1时，返回false；正确插入元素时返回true
